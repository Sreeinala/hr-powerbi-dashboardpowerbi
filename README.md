# HR Analytics Dashboard

## 📊 Overview
This project analyzes employee attrition and workforce trends using an interactive dashboard.

## 🚀 Key Features
- Attrition analysis (16.2%) across 489 employees
- Age group and salary slab segmentation
- Job satisfaction analysis by role
- Department-level filtering (HR, Sales, R&D)

## 🛠 Tools Used
- Power BI
- DAX
- Excel

## 📈 Key Insights
- Highest attrition in age group 26–35
- Majority attrition in salary < 5K
- Sales Executive and Lab Technician roles show higher attrition
